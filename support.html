Frequently Asked Questions

Is my data saved between sessions?

Not currently. Strive is a static dashboard — task checkboxes and timer state reset when you close the page. This is a planned feature.

Can I add my own tasks?

Not yet. Tasks are currently hardcoded into the dashboard. We plan to add editable tasks in a future update — submit a feature request if this is important to you.

The timer doesn't count down. Is that a bug?

The timer is a visual placeholder for now. Interactive timer functionality is on the roadmap. Use the Pomodoro method manually in the meantime.

How do I log a study session?

Head to the Log page to view your session history and weekly reflection. Manual logging is coming in a future release.
